本地mac测试老的minio(最后一个没有用Erasure Coding的版本, 文件存在物理系统的，没有分片存储)  docker运行命令:
    # 最重要的权限和物理机目录测试成功------
    # 运行在前端, 结束后自动删除容器
    docker run --rm -p 9000:9000 -p 9001:9001 \
    -v /Users/yuanxiao/workspace/0tmp/minio_data:/data \
    minio/minio:RELEASE.2022-05-26T05-48-41Z server /data --console-address ":9001"

    # 运行在后台
    docker run --rm -p 9000:9000 -p 9001:9001 \
    -v /Users/yuanxiao/workspace/0tmp/minio_data:/data \
    -d minio/minio:RELEASE.2022-05-26T05-48-41Z server /data --console-address ":9001"
