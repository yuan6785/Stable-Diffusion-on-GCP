mac二进制下载的服务器版本:
    使用以下命令在 macOS 上下载并运行独立的 MinIO 服务器。替换/data为您希望 MinIO 存储数据的驱动器或目录的路径。
    mkdir ./data

    wget https://dl.min.io/server/minio/release/darwin-amd64/minio.RELEASE.2023-04-20T17-56-55Z
    chmod +x minio
    ./minio -h  # 可以查看帮助
    ./minio server -h  # 可以查看具体服务器启动的帮助
    MINIO_STORAGE_CLASS_STANDARD=1&&./minio server minio_data
    -----
